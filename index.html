Website Description

Just a website to see if I could make one, it works best on a desktop computer.
Got a couple high throughput screening data analysis tools on the other tabs.
Grid Plate is if you have multiple plate maps already in excel and want to combine them.
Control Selection is for selecting controls on a plate using the mouse.
Any Selection is where you can input any value and select where it is on a plate using the mouse.
