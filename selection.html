Control Selection Tool

1. Click on the button for the plate size you want to load (96, 384, 1536)

2. Choose the control label to add (Empty will remove labels for selected wells)

3. On the plate map select areas and the control will automatically be applied

4. Repeat steps 2 and 3 until all control data has been added

5. Click Save Data to download as a .csv
